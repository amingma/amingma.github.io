Ultimate Tic Tac Toe

Github Repository

About the Project

This program allows you to play ultimate tic tac toe either against a friend or against the computer. Each turn, all of your possible moves are highlighted. Additionally, the computer uses the alpha beta pruning algorithm to determine its moves. However, the reward function is not super well defined, so the average player should be able to beat the computer.

I built this project because I was learning the basics of front end languages and wanted to apply them to make something fun. I originally was going to build vanilla tic tac toe, but I pivoted to the ultimate version because it was a lot more interesting to think about and implement.

Technologies Used

  • HTML/CSS: Add initial elements and style to page
  • JavaScript: Generate 9x9 game board, add responsiveness to gameplay, provide CPU logic
