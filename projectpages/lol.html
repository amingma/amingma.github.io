About the Project

This application allows you to track you and your friends' progress through the current League of Legends ranked season. You have the option to add friends into the table, delete them, and refresh their information. Additionally, you can click into a friend's profile to view their most recently played matches and champions.

I built this project because I was learning about React and RESTful APIs, and wanted to utilize my knowledge in the two areas to create a simpler full stack application. I occasionally dabble in League, and, fortunately, the parent company Riot has a whole API for developers to use. As a result, the idea came pretty naturally to create an app that integrated the Riot API to display various information to the user.

Technologies Used

  • React: Add elements to page, respond to user input accordingly, fetch data from backend
  • Flask: Provide endpoints to retrieve information from Riot API
  • CSS: Add minimal styling
